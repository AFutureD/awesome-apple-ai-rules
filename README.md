## Contributing

### File Rules

1. Use `[INFOMATION]` to warp information if it can be change under different projects.
2. Use `` ` `` to wrap code.
3. Use `@` as the file path prefix, such as `` `@./AGENTS.md` ``
4. Use `{{ARGUEMENT}}` as contenx argument, such as `{{APP PRODUCT IDENTIFILER}}`
